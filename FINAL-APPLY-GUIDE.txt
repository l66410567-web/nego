빛고을장례119 메인 Hero 최종 통합본

수정 파일
1. index.html
2. style.css

적용 방법
- GitHub 저장소의 기존 index.html과 style.css를 이 파일로 교체합니다.
- bg-hero.jpg는 기존 저장소 루트 파일을 그대로 사용합니다.
- script.js는 기존 상담 모달 함수 openModal()을 그대로 사용하므로 변경하지 않았습니다.

주요 반영
- 확정된 Hero v4 디자인 실제 메인페이지 통합
- 주요 CTA를 기존 상담 모달(openModal)과 연결
- 보조 CTA를 실제 상담 예약 영역(#apply)과 연결
- 모바일 순서 최적화
- 배경 이미지 preload 적용
- 메타 설명, Open Graph, 검색 키워드 보완
- 중복되던 기존 알림 띠 제거
- 지원 범위 조건 문구 유지

배포 후 확인
- 첫 화면 배경 이미지 표시
- '받을 수 있는 지원 확인하기' 클릭 시 상담 모달 열림
- '상담 예약' 클릭 시 신청 영역으로 이동
- 모바일 360/390/430px 화면에서 버튼과 제목 잘림 없음
